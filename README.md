# Portafolio

Página de portafolio personal de un desarrollador web. En una sola página reúne
una breve presentación profesional, las habilidades y herramientas que se
manejan, una selección de proyectos destacados y una sección de contacto.

## Funcionalidad

- **Encabezado / Presentación:** nombre, rol profesional y enlaces de acción
  (descargar CV, ir a GitHub y contacto).
- **Habilidades y herramientas:** listado visual de las tecnologías que se dominan.
- **Proyectos destacados:** tarjetas con los proyectos más representativos y su
  descripción.
- **Contacto:** correo electrónico y enlaces a redes profesionales (LinkedIn y
  GitHub).
- **Diseño responsivo:** la página se adapta a distintas resoluciones (escritorio,
  tablet y móvil).

## Tecnologías y técnicas

- **HTML5** semántico (`header`, `main`, `section`, `footer`, `ul`/`li`).
- **CSS3** con **Flexbox** para la maquetación y **media queries** para el diseño
  responsivo.
- **Metodología BEM** para nombrar las clases y organizar los estilos por bloques
  (`blocks/`, `pages/`, `vendor/`).
- **normalize.css** para unificar los estilos base entre navegadores.
- Tipografías **Open Sans** y **Archivo Black**, con fuentes del sistema como
  alternativa.
